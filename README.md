# Hi, I'm Lin 👋

Building small, sharp tools that automate the boring parts of running a one-person business.

## What I'm shipping

- **[voice2cc](https://github.com/lfzds4399-cpu/voice2cc)** — Push-to-talk for Claude Code. Hold a key, speak, get text. Cuts the typing tax on long sessions.
- **[domain-harness](https://github.com/lfzds4399-cpu/domain-harness)** — Automated domain-investing pipeline. Generates candidates, scores them, checks availability, drafts listings.

## Stack

Python · TypeScript · Next.js · Claude Agent SDK · Remotion · FastAPI

I work in a *harness* style — small CLI-driven pipelines with `agents/`, `validators/`, and a `manifest.yaml`. Composable, observable, easy to bisect when something breaks.

## Working in public

Everything I open-source is something I actually use daily. No abandoned demos, no half-finished frameworks.

If a repo is here, I'm running it on my own machine right now.
